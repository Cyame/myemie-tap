# Myemie-tap

写给咩咩的生日惊喜  ——茶目

### 点击进入

[Mietap](mietap.cyame.club)
### 这是什么

一个简单好玩的咩咩琴，可以通过在页面上滑动鼠标触发区块来弹出节奏明快的乐曲。

*Just Enjoy it!*
